Form Generation

Form is generated dynamically based on whatever item is passed to it. Fields are assigned a "field directive based on data type and validation parameters. New entries are assigned a range of fields based on item type (try switching between types when creating a new item).

Editting

Fields are edittable by clicking "edit" button. Validation rules prevent saving if errors. Tracks are re-orderable via drag & drop
